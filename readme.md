# A real chat application

## How to use

- install dependencies `npm install` <br />

- start project `npm run dev`

## Technologies

- react v18
- firebase v9
- vite
- heroicons
- contextAPI

## Features

- realtime chat and replies
- sharing of media files
- online status
- unread notifications
- last chat preview
- set profile picture
- delete profile picture
- authentication
